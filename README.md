# New EDVKONZEPTE

Statische Landingpage fuer den neuen Auftritt von `K&N EDV Konzepte`.

## Dateien

- `index.html` enthaelt die Seitenstruktur.
- `styles.css` enthaelt das komplette visuelle System.
- `script.js` steuert Splash-Screen und Reveal-Animationen.
- `assets/` enthaelt Logos, Portraits und Bilder.

## Deployment

Die Seite ist bewusst als statische Website angelegt:

- direkt geeignet fuer `GitHub + Vercel`
- spaeter leicht auf einen eigenen VPS uebertragbar
- keine Baukasten-Abhaengigkeit
